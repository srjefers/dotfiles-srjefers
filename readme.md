# DotFiles by Sr. Jefers
Archivos de configuracion, usados en I3WM, sobre Debian Strech 9.
El archivo `Xresources` controla los estilos.

## Aplicaciones usadas
* I3-gaps
* Telegram
* HTOP
* Ranger
* Rofi
* tty-clock
* Urxvt-unicode [terminal por defecto]
* Cowsay
* Polybar
* Neofetch
